=========================================
Welcome to the Devilry user documentation
=========================================

User documentation for `Devilry <http://devilry.org>`_.

.. note::
    You can contribute to these docs. Use the *Show on GitHub* or *Edit on
    GitHub* (if logged into github) links in the sidebar.

    You can also submit issues with these docs to
    `the devilry-userdoc issue tracker <https://github.com/devilry/devilry-userdoc/issues>`_.


Getting started
###################
We recommend that you start with a quick look at :ref:`commonconcepts`.
This avoids confusion with a few special terms that Devilry uses to allow
it to be used in many different settings. When you have skimmed over the concepts,
you can continue with the other Topic guides below.
